#API de Filmes da Marvel

API REST dedicada ao Universo Cinematográfico Marvel (MCU). Esta API permite consultar informações sobre filmes, personagens, heróis e vilões presentes no universo Marvel. Também é possível cadastrar novos filmes e acessar detalhes completos sobre cada produção.

##  Endpoints disponíveis

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /filmes | Lista todos os filmes da Marvel |
| POST | /filmes | Cadastra um novo filme |
| GET | /filmes/{id} | Busca um filme específico pelo ID |
| GET | /filmes/{id}/personagens | Lista os personagens de um filme |
| GET | /personagens | Lista todos os personagens |
| GET | /personagens/{id} | Busca um personagem específico pelo ID |
